# Sito HTML — Cinema Sperimentale LUMEN

Sito vetrina statico in HTML5 + CSS3, composto da 5 pagine:

- `index.html` — Home / presentazione
- `programmazione.html` — Film in sala con orari
- `info.html` — Chi siamo, sala, dove siamo
- `contatti.html` — Recapiti e form contatti
- `prenotazione.html` — Pagina con link all'applicativo TypeScript

## Avvio
Aprire `index.html` direttamente nel browser. Nessuna build richiesta.

## Documentazione
`doc_sito.pdf` — alberatura e descrizione del programma.
